# IAMYM Landing Page

Landing estática para IAMYM Creator Network.

## Archivos
- `index.html`
- `style.css`
- `script.js`
- `assets/iamym-logo.jpeg`

## Cómo verla
Abrí `index.html` en el navegador.

## Cómo subirla online
Podés subir la carpeta a Netlify, Vercel o GitHub Pages.

## Cambios rápidos
- Cambiar mail: buscá `iamymnetwork@gmail.com`.
- Cambiar X: buscá `@_iamym` o `https://x.com/_iamym`.
- Cambiar textos: editar `index.html`.
